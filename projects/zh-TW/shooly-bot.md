
# Shooly：對話式自動化服務

**Shooly** 是一款智慧型 LINE 聊天機器人，旨在優化都會區居家清潔與維修服務的預約流程。

## 專案目標

居家服務產業常面臨溝通碎片化的問題。我們的目標是打造一個「零介面」的體驗，讓使用者能在他們每天使用的平台上完成預約、支付與進度追蹤。

## 技術實作

- **自然語言理解 (NLU)**：使用 Python 與 Spacy 建立自定義意圖分類模型，處理非正式且口語化的使用者輸入。
- **狀態機架構**：採用強健的後端狀態機管理複雜的多步驟預約流程，有效防止使用者流失。
- **LINE Messaging API**：充分利用 Rich Menus 與 Flex Messages，在文字框中打造類似應用程式的圖形介面體驗。

## 成果

Shooly 在 **LINE Chatbot 對話機器人設計大賽** 中榮獲 **第一名 (冠軍)**，評審一致認為其在技術深度與市場實用性之間取得了卓越平衡。
